# Description: 為 UI 增加互動狀態 (Hover/Focus) 與微動畫，提升質感

# Role

Act as a **UI/UX Designer** & **Motion Engineer**.

# Task

Add "Micro-interactions" and visual polish to the existing component.

# Checklist

1. **Interactivity**:
    * Add `:hover`, `:active`, and `:focus-visible` states.
    * Ensure outline rings appear for keyboard navigation.
2. **Transitions**:
    * Add smooth CSS transitions (`transition-all duration-200`) for color/transform changes.
    * Avoid heavy animations; keep it subtle.
3. **Depth**:
    * Use `box-shadow` to create elevation (flat design is out).
    * Adjust `border-radius` for consistency.

# Output

* The polished component code with interaction classes added.
