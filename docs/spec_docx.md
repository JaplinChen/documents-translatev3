# DOCX 翻譯規格

## 1. 目標
翻譯 Word 文件 (.docx)，並保留段落樣式、標題階層、表格結構。

## 2. 範圍
- **段落 (Paragraphs)**：保留字型大小、粗體、斜體。
- **表格 (Tables)**：保留表格佈局，翻譯儲存格內文字。
- **清單 (Lists)**：保留項目符號與編號。

## 3. 資料模型
- `block_type`: `paragraph` | `table_cell`
- `source_text`: 段落文字內容
- `style`: 樣式名稱或屬性（供套用時參考）

## 4. 品質要求
- 分頁符號應盡可能保留。
- 翻譯後的段落應套用與原文一致的 Style。
