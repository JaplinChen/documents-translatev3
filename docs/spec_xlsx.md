# XLSX 翻譯規格

## 1. 目標
翻譯 Excel 活頁簿 (.xlsx)，保留儲存格格式、字型樣式、邊框與數值格式。

## 2. 範圍
- **工作表 (Worksheets)**：遍歷所有分頁。
- **儲存格 (Cells)**：翻譯文字型儲存格，略過公式（或僅翻譯公式產出的靜態文字）。
- **佈局**：維持原有的欄寬與列高。

## 3. 資料模型
- `sheet_name`: 工作表名稱
- `cell_address`: 例如 `A1`, `B12`
- `block_type`: `excel_cell`
- `source_text`: 儲存格內容

## 4. 品質要求
- 數字、日期與公式結構不可被破壞。
- 對於長文字，需支援「自動調整列高」或「文字折行」設定。
